# 📊 Social Media Dashboard

A full-featured Social Media Dashboard built using **Django** and **Python**, connected with **Firebase** and a manually configured **SQLite** database. This project integrates authentication via **Google Developer Console** and **Facebook Developer Console**, allowing users to log in using their social accounts.

## 🚀 Features

- 🔒 Social Login via Google and Facebook
- 📸 Post creation with images, descriptions
- 💬 Interactions: Like, Comment, and Share
- 🧑‍💼 User-specific posts and profile display
- 🗃️ Data stored in both SQLite (local) and Firebase
- 📱 Clean, responsive UI

---

## 🛠️ Tech Stack

- **Backend**: Django (Python)
- **Database**: SQLite (local) + Firebase (cloud)
- **Authentication**: Google & Facebook OAuth via Developer Consoles
- **Frontend**: HTML, CSS, JS (via Django templates)
